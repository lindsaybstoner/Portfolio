PORTFOLIO

I am a current student of the Penn LPS Coding Bootcamp. Each of these projects have been created for the coding bootcamp to showcase my coding ability. All projects can be found on my Github Profile.

Stroll - Full Stack Application

My group and I created an app that allows you to login and create profiles for your dogs. From there you can find tailored information to each of your dogs such as how long have your walks been, where they have met dog friends, and where there are free doggie bags around the city.

Burger App - Full Stack Application

I created a burger logger with MySQL, Node, Express, Handlebars and a homemade ORM. It lets the user input the names of burgers they'd like to eat, then eat them, and lastly throw out those burgers out.

Friend Finder App - Front End Application

I built a compatibility-based "FriendFinder" application. It takes in results from your users' surveys, then compare their answers with those from other users and then gives you your best match.

BAmazon App - Back End Application

I built an Amazon-like storefront with MySQL in the command line. The app takes in orders from customers and deplete stock from the store's inventory.

stayTuned App - Front End Application

My group and I created an app to connect users to musicians they like and discover more about that specific artist. I focused on calling the APIs such as BandsInTown, Lyrics.OVH, Wikapedia, and iTunes.
